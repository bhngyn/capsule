Pair with Capsule

  1. Open Capsule in your browser (default http://localhost:8080).
  2. Go to Settings → Browser extension and click Pair a new extension.
  3. Copy the token Capsule shows you (it's revealed once).
  4. Paste it below.

Once paired, the extension talks only to the Capsule server URL above. It never sends data to the public internet. The token is stored in your browser's extension storage and is never shared with web pages.
